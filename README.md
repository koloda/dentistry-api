#Installation

[//]: # (ssh command for copy .env from .env.example)
```bash
cp .env.example .env
```

[//]: # (ssh command for install composer)
```bash
composer install
```

[//]: # (ssh command for generate key)
```bash
php artisan key:generate
```

[//]: # (ssh command for migrate database)
```bash
php artisan migrate
```
